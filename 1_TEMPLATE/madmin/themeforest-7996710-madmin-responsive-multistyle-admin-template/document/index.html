1. Overview

μAdmin is The Ultimate Responsive Admin Template.

μAdmin built on Bootstrap 3.1 Framework with full responsive technique. It's a multiform template with two style and more than 20 color themes, so you can use μAdmin for many types of web applications like custom admin panel, admin dashboard, application backend, etc.

μAdmin have many kind of Transition effects, UI Element and components to make your application excited and professional. Beside it, Charts option included will provide many ideas for you to update status of business continuously.

Enjoy using μAdmin and stay tuned information. If you have any suggestion, contact with us admin@swlabs.co

New features will be coming soon in future.
